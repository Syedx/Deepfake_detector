Our Technology

Leveraging cutting-edge deep learning to detect manipulated media

CNN Architecture

Our system uses a Convolutional Neural Network (CNN) based on ResNet50 architecture, trained on thousands of authentic and manipulated images to recognize subtle patterns of manipulation.

The deep learning model has been fine-tuned specifically to detect GAN-generated faces, face swaps, and other common deepfake techniques with high accuracy.

Metadata Analysis

Beyond visual analysis, our system extracts and examines EXIF data from images, looking for inconsistencies or signs of editing that might indicate manipulation.

This metadata analysis provides additional context to the AI's visual assessment, creating a more comprehensive evaluation.

Grad-CAM Visualization

Our system generates heatmaps using Gradient-weighted Class Activation Mapping (Grad-CAM) technology to highlight the regions of the image that most influenced the detection decision.

This transparency helps users understand why an image was classified as real or fake, providing insight into the AI's decision-making process.
